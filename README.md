# 🚀 TechPath

TechPath is a modern platform designed to help students and aspiring professionals discover, plan, and track their technology learning journey. It provides structured roadmaps, curated learning resources, project recommendations, and progress tracking to help users build industry-ready skills.

---

## 📌 Overview

Choosing the right technology stack and learning path can be overwhelming. TechPath simplifies this process by providing clear roadmaps for various technical domains along with resources and project ideas that guide users from beginner to advanced levels.

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/saravana5632/techpath.git
```

### Navigate to the project

```bash
cd techpath
```

### Install dependencies

Frontend

```bash
cd client
npm install
```

Backend

```bash
cd server
npm install
```

### Start the application

Backend

```bash
npm start
```

Frontend

```bash
npm run dev
```
